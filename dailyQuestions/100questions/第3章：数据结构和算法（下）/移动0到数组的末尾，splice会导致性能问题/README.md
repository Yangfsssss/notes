将数组中的0移动到末尾：
    如输入[1,0,3,0,11,0]，输出[1,3,11,0,0,0]；
    只移动0，其他顺序不变；
    必须在原数组进行操作；

    如果不限制“必须在原数组操作”：
        定义part part2两个数组；
        遍历数组，非0 push到part，0 push到part2；
        返回part.concat(part2);

    传统思路：
        遍历数组，遇到0则push到数组末尾；
        用splice截取掉当前元素；
        时间复杂度是O(n^2) --- 算法不可用；

    使用双指针：
        定义i指向第一个0，j指向i后面的第一个非0；
        交换j和i的值，继续向后移动；
        只遍历一次，所以时间复杂度是O(n)；

划重点：
    向面试官确认：是否必须修改原数组？
    数组是连续存储，要慎用splice unshift等API；
    双指针思路解决嵌套循环问题；
